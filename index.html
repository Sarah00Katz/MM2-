About

The Magic Mirror is a framed 24” x 36” Interactive LED screen with a raspberry pi computer and webcam. This screen is a self contained unit that contains a weather proof frame, requires a single power outlet and allows for artworks created in processing to be displayed.

Process

Gray Area Foundation for the Arts and NOW Hunters Point came together to create a 10-week Youth Creative Code Apprenticeship program. These youth were taught the skinny necessary for hacking, coding, and soldering creative projects together. Over the course of 10 weeks, from October to December 2016, Teenagers worked alongside Creative Code Artists to develop an interactive artwork public viewing. The team used Processing, an open sourced language for visual programming, and Makey Makey, for visual computing.
